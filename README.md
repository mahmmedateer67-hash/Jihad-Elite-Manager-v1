# Jihad Manager - Premium Edition (Private)

هذا المستودع يحتوي على النسخة المؤمنة والمشفرة من مشروع Jihad.

## المكونات:
- `Jihad.sh`: السكريبت الرئيسي (الواجهة والبروتوكولات).
- `ionos_dns_manager_obfuscated.py`: وحدة إدارة DNS مشفرة لحماية مفاتيح API.
- `pyarmor_runtime_000000/`: مكتبات التشغيل اللازمة للكود المشفر.
- `bin/`: الملفات الثنائية (Binaries) اللازمة للبروتوكولات.

## التشغيل:
1. تأكد من وجود كافة الملفات في نفس المجلد.
2. أعط صلاحيات التشغيل: `chmod +x Jihad.sh bin/*`
3. ابدأ المشروع: `./Jihad.sh`

**ملاحظة أمنية:** تم تشفير الكود لحماية مفتاح IONOS API الخاص بك. لا تقم بمشاركة هذا المستودع مع أي شخص.
